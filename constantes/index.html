constantes

Regras para criar constantes:

  • Não se pode criar constantes com palavras resrvadas
  • Precisam ter nomes significativos
  • Não pode comecar nome de constantes com numero
  • Não podem conter espaços
  • Utiliza-se camelcase
  • São case sensitive
  • Não podemos modificar o valor de uma constante
  • Não utilize var, utilize const
  • Deve ser criada e inicilizada
